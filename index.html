100% 雲端自動追蹤部署指南

本專案採用 Serverless 無伺服器架構，所有監控與通知皆在 GitHub 雲端運行，不需要在您的電腦下載或運行任何程式。

1

更新檔案至 GitHub 倉庫

將修改後的 scraper.py, index.html, 和 app.js 檔案，直接上傳或複製貼上到您現有的 GitHub 倉庫中並儲存。

2

設定 Telegram 憑證 (GitHub Secrets)

如果還沒有設定，請至 Settings -> Secrets and variables -> Actions 新增：

  • TELEGRAM_BOT_TOKEN：您的 Telegram Bot Token
  • TELEGRAM_CHAT_ID：您的個人 Telegram ID (一串純數字)

💡 備註：已經不需設定 PRICE_THRESHOLD 限制，系統會全自動依照「日本各航線的平均行情」算出特價區間！

3

手動執行以初始化資料

上傳完成後，至倉庫的 Actions ➔ Scheduled Flight Price Tracker ➔ 點選 Run workflow 手動執行一次爬蟲，完成後重新整理您的 Pages 網頁即可！
