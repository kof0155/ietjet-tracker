100% 雲端自動追蹤部署指南

本專案採用 Serverless 無伺服器架構，所有監控與通知皆在 GitHub 雲端運行，不需要在您的電腦下載或運行任何程式。請按照以下簡單的 3 個步驟進行設定，即可在您的手機上建立自動票價監控系統！

1

建立並上傳程式碼至 GitHub 倉庫

在您個人的 GitHub 帳號中，建立一個新的儲存庫 (Repository)，並將本目錄下的所有程式碼上傳（建議設為 Private 私人倉庫 確保您的設定隱私）：

git init
git add .
git commit -m "Initialize VietJet Tracker"
git branch -M main
git remote add origin https://github.com/您的帳號/儲存庫名稱.git
git push -u origin main
2

設定 Telegram 憑證 (GitHub Secrets)

為了讓雲端爬蟲能夠發送 Telegram 訊息給您，請在 GitHub 儲存庫的 Settings -> Secrets and variables -> Actions 中，新增以下三個 Repository Secrets：

  • TELEGRAM_BOT_TOKEN：您的 Telegram Bot API Token (跟 @BotFather 建立 Bot 即可獲得)
  • TELEGRAM_CHAT_ID：您的個人 Telegram ID (向 @userinfobot 發送任意訊息即可取得)
  • PRICE_THRESHOLD：您的目標警報門檻價格，例如 10500
3

啟用 GitHub Pages 網頁儀表板

上傳程式碼並設定金鑰後，將本儀表板發布為公開網頁，以便您隨時查看票價圖表：

  1. 在 GitHub 儲存庫的 Settings -> Pages
  2. 在 Build and deployment -> Source 選擇 Deploy from a branch
  3. 在 Branch 選擇 main 並指定 / (root)，然後點選 Save
  4. 數分鐘後，您即可透過 https://您的帳號.github.io/儲存庫名稱/ 打開本儀表板！
🎉

完成！自動監控啟用中

GitHub Actions 將會每 6 小時自動在雲端運行一次爬蟲。當票價低於您設定的門檻值時，您的 Telegram 會即時收到推播通知！您也可以在 GitHub 儲存庫的 Actions 分頁點選 Run workflow 來手動立刻測試抓取。
